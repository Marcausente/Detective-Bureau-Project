# Detective Bureau Project

Pagina WEB dedicada a la gestión y control de la Detective Bureau de la LSPD, desarrollado para FiveM, con ella, podremos realizar cualquier tipo de accion relacionada con esta rama, suponiendo un control absoluto en todos estos sistemas.

- Mapa de casos y bandas, incluyendo un mapa publico de acceso general a la policia y un mapa privado solo para detectives
- Control completo de persona y roles
- Control y gestion de casos, con utilidades completas
- Control y gestion de organizaciones criminales
- Caracteristicas de perfil completas
- Tablon de anuncios y novedades
- Archivo de interrogatorios
- Archivo de incidentes y salidas
- Archivo de documentación interna y recursos

Todas estas caracteristicas estan controladas por una base de datos y ligadas entre ellas, con la posibilidad de atar una cosa con otra sin ningun tipo de restricción, creando así el sistema de gestión de detectives mas complejo y completo existente hasta la fecha, permitiendonos hacer cualquier cosa, seguir cualquier caso sin interrupciones.
